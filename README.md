# 📖 Sobre o projeto

O **ListFlow** é uma aplicação desenvolvida em **React** para gerenciamento de tarefas. O usuário pode adicionar e remover tarefas, visualizar seus detalhes em uma página dedicada e contar com persistência dos dados por meio do **Local Storage**, mantendo as informações salvas mesmo após atualizar a página.

Este projeto foi desenvolvido com foco em aprendizado prático, aplicando conceitos fundamentais do ecossistema React e boas práticas de desenvolvimento Front-end.

---

# ✨ Funcionalidades

* ✅ Adicionar tarefas
* 🗑️ Remover tarefas
* 📄 Visualizar detalhes de uma tarefa em uma página específica
* 💾 Persistência das tarefas utilizando Local Storage
* ⚡ Navegação entre páginas com React Router
* 📱 Interface responsiva desenvolvida com Tailwind CSS
